# AMCA Safety Hub — Prototype

A clickable prototype of a web-native safety manual for AMCA: task-based navigation,
a full section index (00–10), plain-language document pages, search, and a
navigation assistant.

Built by ARM Hub × Aether.

## View it
Open `index.html` in any browser, or visit the published GitHub Pages URL.

## Notes
- Single self-contained HTML file — no build step, no dependencies.
- The brand colour is a single token (`--brand`) at the top of the stylesheet;
  drop in the official AMCA hex to make it pixel-exact.
- Sample content only; not the live manual.
